## 📝 프로젝트 소개

_가격 걱정 없이 합리적인 가격으로 득텔 하세요!_

사기매물과 과도한 리셀 프리미엄으로 인해 신뢰도가 낮았던 숙소 양도거래를 혁신합니다. 국내 최대 숙박 플랫폼 야놀자에서 인증이 된 상품만을 취급합니다. 또한 과도한 프리미엄이나 흥정을 불가능하며 사용하지 못할 상품을 원하는 가격에 지정하고 N시간 안에 안 팔리면 2차 할인 가격으로 다운되는 구조를 가지고 있습니다.

## ✔️ 야놀자 RFP 구현 사항

- [x] 회원은 회원가입을 할 수 있습니다.

- [x] 기본 정보는 ID 역할로 이메일 주소를 사용하고, 그 외에 비밀번호와 이름입니다.

- [x] 데이터베이스에는 암호화된 비밀번호가 들어가야합니다.

- [x] 이메일과 비밀번호로 로그인할 수 있습니다.

- [x] 회원 정보를 저장해둔 데이터베이스를 검색하여 해당 사용자가 유효한 사용자인지 판단합니다.

- [x] 로그인을 하지 않아도 상품을 확인할 수 있습니다.

- [x] 회원은 판매자와 구매자 모두 가능합니다.

- [x] 지역/날짜별 양도 상품을 확인할 수 있습니다.

- [x] 원래 구매금액과 양도 상품의 금액을 모두 확인할 수 있습니다.

- [x] 구매한 상품의 가격 정보를 확인할 수 있습니다.

- [x] 예약 상품을 상세하게 확인하고 이를 기반으로 체크인을 안전하게 완료할 수 있어야 합니다.

- [x] 내가 구매했으나 양도할 상품을 확인하고 선택할 수 있어야 합니다.

- [x] 양도 완료 이후 판매금액, 수수료금액 등 관련 정산 정보를 확인할 수 있어야 합니다.

- [x] 양도 상품의 희망 거래가격을 설정할 수 있어야 합니다.

- [x] 안전한 C2C 거래 결제 시스템을 제공해야 합니다.

┃  ┣ ...

## 📍 컨벤션

| 커밋 컨벤션 |                                                             |
| ----------- | ----------------------------------------------------------- |
| feat        | 새로운 기능 추가                                            |
| fix         | 버그 수정                                                   |
| style       | 코드 스타일 수정 (세미 콜론, 인덴트 등의 스타일적인 부분만) |
| design      | css 등 디자인 추가 및 수정                                  |
| refactor    | 코드 리팩토링                                               |
| comment     | 주석 추가/수정                                              |
| docs        | 내부 문서 추가/수정                                         |
| test        | 테스트 추가/수정                                            |
| chore       | 빌드 관련 코드 수정                                         |
| remove      | 파일 삭제                                                   |
